## Eval for hardened Success Criterion 3
Run one representative unattended kanban phase (e.g. Phase 5 review, autonomous mode) under the profile, capture the transcript/event stream, and FAIL on any of:
- blocked stdin read / `input()` wait
- "should I", "do you want", "confirm?", "proceed?" language
- narration exceeding a token budget for a successful phase
- missing final artifact / missing state transition at phase end
Baseline: same phase under the default profile, to show the delta the profile buys.
